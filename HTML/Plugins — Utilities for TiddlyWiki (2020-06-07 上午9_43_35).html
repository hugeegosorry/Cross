Free and open source plugins for TiddlyWiki version 5.1.22 or newer

More – but older – plugins are still available: Visit https://tid.li/tw5/plugins-2019.html to check them out. Most of them will be updated and moved over here when I find the time.

These plugins were made to improve my workflow, the design and the functionality of my wikis. – Enjoy!

Thomas Elmiger

  2. Drag the title or the icon below to your wiki.

DetailsWidget 0.7.5

Create HTML 5 <details> elements including a <summary> using this widget.

The details element – W3C definition

The details element represents a disclosure widget from which the user can obtain additional information or controls.

W3C
Code example
<$details summary="This should be open" open="yes">

Content will be immediately visible if open is set to "yes".

</$details>

EditButtons 1.0.6

The EditButtons plugin makes editing more efficient by adding three buttons to the edit toolbar:

Cancel & Close (Discard & Close)

Save & Close (Done & Close)

Save & Keep Open (Done & Reopen)

Inspired by Josiah and Richard William Smith. See Discussion in the Google Group.

Dependencies: –

EditorCounter 0.6.6

The EditorCounter plugin can display counters for words and characters in the edit view. In single page wikis it can save texts automagically in the background – check the plugin settings.

Dependencies: –

HarveyBalls 0.0.6

Round ideograms used for visual communication of qualitative information. – The set includes icons with titles and descriptions as well as handy macros for display.

Dependencies: –

PluginSize 0.1.1

This plugin calculates the size of all installed plugins, including themes and languages. It adds a tab Plugin Info to the $:/ControlPanel – the one with the icon.

Dependencies:

$:/plugins/telmiger/EditorCounter
$:/plugins/telmiger/HarveyBalls
$:/plugins/telmiger/rpn

rpn 0.7.3

rpn is a lightweight Javascript macro for basic math operations. It is inspired by formerly popular calculators using Reverse Polish Notation (see Wikipedia for more info). Usage example:

<<rpn 5000 200 * 2 p en-GB>> output: 1,000,000.00

Dependencies: –

Simple Search 1.0.24

A nice and simple overlay interface for searching your wiki. Designed for mobile and desktop screens. Reduce to the max. The standard search input field in the sidebar is removed, instead there is a button.

Dependencies: –
